Em JavaScript, uma **variável** é como uma caixa que você usa para guardar um valor. 
Você dá um nome a essa caixa para poder se referir a ela e usar o que está dentro dela mais tarde.

Por exemplo, você pode criar uma variável chamada `idade` e colocar o número `30` dentro dela.

Existem três palavras-chave principais para declarar variáveis em JavaScript:

* **`var`**: É a forma mais antiga. As variáveis declaradas com `var` têm um escopo de função, 
o que pode causar alguns comportamentos inesperados.

* **`let`**: É a forma moderna e preferida. As variáveis declaradas com `let` têm um escopo de bloco,
 o que é mais previsível e seguro.

* **`const`**: Também moderna e usada para valores que não mudam. Uma vez que você atribui um valor a uma variável `const`,
 você não pode reatribuí-lo.

let e const para declarar variaveis.

Exemplo : 

let nome = "Matheus"
const idade = 31;

let defini váriavies em escopo global e const variavel de escopo private.
